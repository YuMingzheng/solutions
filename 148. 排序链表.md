排序链表

### 解题思路

#### 方法一

使用`PriorityQueue`

```java
class Solution {
    public ListNode sortList(ListNode head) {
        PriorityQueue<ListNode> pq = new PriorityQueue<>((n1,n2)->Integer.compare(n1.val,n2.val));
        ListNode ptr = head;
        while(ptr != null){
            pq.offer(new ListNode(ptr.val));
            ptr = ptr.next;
        }
        ListNode dummy = new ListNode();
        ListNode ptrD = dummy;
        while(!pq.isEmpty()){
            ptrD.next = pq.poll();
            ptrD = ptrD.next;
        }

        return dummy.next;
    }
}
```

**算法复杂度分析：**

时间复杂度：$O(nlogn)$

空间复杂度：$O(n)$
