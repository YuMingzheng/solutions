### 问题描述

[题目链接](https://leetcode.cn/problems/jump-game/?envType=study-plan-v2&envId=top-100-liked)

数组nums中的每一个值表示在这个位置能够条约的最大格子数，问跳到数组末尾的最小步数

### 解题思路

#### 方法一

- 维护当前能到的最右端curr_right
- 遍历数组nums
	- 更新当前可以到的最右端next_right=max(next_right, i + nums[i])
	- 判断当前是否到了最右端curr_right
		- 如果到了，则更新curr_right
		- ans++

```python
class Solution:
    def jump(self, nums: List[int]) -> int:
        ans = 0
        curr_right = 0
        next_right = 0

        for i in range(len(nums)-1):
            next_right = max(i + nums[i], next_right)
            if i == curr_right:
                curr_right = next_right
                ans += 1
        return ans
```

时间复杂度：$O()$

空间复杂度：$O()$
