### 问题描述

[题目链接](https://leetcode.cn/problems/minimum-size-subarray-sum/description/)

给定一个含有 n 个正整数的数组和一个正整数 s ，找出该数组中满足其和 ≥ s 的长度最小的 连续 子数组，并返回其长度。如果不存在符合条件的子数组，返回 0。

### 解题思路

#### 方法一

使用滑动窗口，右指针`ptr_r`先遍历，找到从0到ptr_r的和恰好超过target的index
此时sum>target，一点点收缩左指针`ptr_l`，直到sum< target
右指针继续下一次for迭代

```python
class Solution:
    def minSubArrayLen(self, target: int, nums) -> int:
        res = len(nums)+1
        sum = 0
        ptr_l = 0
        for ptr_r in range(len(nums)):
            sum += nums[ptr_r]
            while sum >= target:
                res = min(res , ptr_r-ptr_l+1)
                sum -= nums[ptr_l]
                ptr_l+=1

        return 0 if ptr_r == len(nums)-1 and ptr_l == 0 and sum < target else res
Solution().minSubArrayLen( nums =[1,1,1,1,1,1,1,1] , target = 11)
```

**算法复杂度分析：**

时间复杂度：$O(n)$

空间复杂度：$O(1)$
