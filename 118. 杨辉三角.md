给定一个整数`numRows`，输出深度为`numRows`的杨辉三角
![[Pasted image 20250401112134.png]]

### 解题思路

#### 方法一

1. 确定dp数组（dp table）以及下标的含义
	1. `List<List<Integer>> dp`，表示结果，也作为递推数组存在
2. 确定递推公式
	1. $dp.get(i).get(j) = dp.get(i-1).get(j-1) + dp.get(i-1).get(j)$
3. dp数组如何初始化
	1. 全设为空或0
4. 确定遍历顺序
	1. for循环从`i=2`开始遍历
		1. for循环从`j=1`开始，到`i`结束
5. 举例推导dp数组

```java
class Solution {
    public List<List<Integer>> generate(int numRows) {
        List<List<Integer>> dp = new ArrayList<>(numRows);
        for(int i = 0 ; i < numRows ; i ++){
            dp.add(new ArrayList<>(i+1));
        }
        dp.get(0).add(1);
        if(numRows == 1)return dp;
        dp.get(1).add(1);
        dp.get(1).add(1);
        if(numRows == 2)return dp;

        for(int i = 2 ; i < numRows ; i ++){
            dp.get(i).add(1);
            for (int j = 1; j < i; j++) {
                dp.get(i).add(dp.get(i-1).get(j-1) + dp.get(i-1).get(j));
            }
            dp.get(i).add(1);
        }
        return dp;
    }
}
```

时间复杂度：$O()$

空间复杂度：$O()$
