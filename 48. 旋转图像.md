### 问题描述

[题目链接](https://leetcode.cn/problems/rotate-image/description/)

把二维数组右旋 90 度。

### 解题思路

#### 方法一

变量 i 用于处理矩阵的每一行，从外到内遍历。当矩阵的中心只有一个元素时，无需进一步处理。变量 j 则负责处理每一列的元素。

```java
class Solution {
    public void rotate(int[][] matrix) {
        int n = matrix.length;
        for(int i = 0; i < n / 2; i++){
            for (int j = i ; j < n-i-1 ; j++){
                int temp = matrix[i][j];
                matrix[i][j] = matrix[n-1-j][i];
                matrix[n-1-j][i] = matrix[n-1-i][n-1-j];
                matrix[n-1-i][n-1-j] = matrix[j][n-1-i];
                matrix[j][n-1-i] = temp;
            }

        }
    }
}
```

**算法复杂度分析：**

时间复杂度：$O(n^2)$

空间复杂度：$O(1)$
