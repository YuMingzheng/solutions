### 问题描述

[题目链接](https://leetcode.cn/problems/jump-game/?envType=study-plan-v2&envId=top-100-liked)

数组nums中的每一个值表示在这个位置能够条约的最大格子数，问能不能从0跳到最后一个格子

### 解题思路

#### 方法一

- 维护“当前能到达的最远位置max_idx”
- 遍历nums中每一个值，判断当前位置是否可以到达 并且是否需要更新最远位置。

```python
class Solution:
    def canJump(self, nums: List[int]) -> bool:
        # 最远的位置
        max_idx = 0
        for i in range(len(nums)):
            if i <= max_idx and  i + nums[i] > max_idx:
                max_idx = i + nums[i]
        return max_idx >= len(nums)-1
```

时间复杂度：$O()$
