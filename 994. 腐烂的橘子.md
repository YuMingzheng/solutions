### 问题描述

[题目链接](https://leetcode.cn/problems/rotting-oranges/description/?envType=study-plan-v2&envId=top-100-liked)

每过一分钟，腐烂的橘子就会传染相邻的好橘子。计算传染所有橘子所需要的最少时间，如果不能全部传染，返回-1；

### 解题思路

#### 方法一

1. **初始化**：
    - 使用队列存储所有初始腐烂的橘子（值为2）。
    - 统计新鲜橘子的数量（值为1）。
2. **BFS遍历**：
    - 每次从队列中取出当前腐烂的橘子，检查其四个方向的相邻单元格。
    - 如果相邻单元格是新鲜橘子（值为1），将其标记为腐烂（值为2），并加入队列。
    - 每次处理完一层（即初始腐烂的橘子及其传播的橘子），如果这一轮有橘子腐烂，则时间加1。
3. **结果判断**：
    - 如果遍历结束后还有新鲜橘子（`freshCount > 0`），返回-1。
    - 否则，返回经过的时间。

```java
public class RottingOranges {
    public int orangesRotting(int[][] grid) {
        int rows = grid.length;
        if (rows == 0) return -1;
        int cols = grid[0].length;
        
        // 方向数组，表示上下左右四个方向
        int[][] directions = {{-1, 0}, {1, 0}, {0, -1}, {0, 1}};
        
        // 初始化队列，存储所有初始腐烂的橘子
        Queue<int[]> queue = new LinkedList<>();
        int freshCount = 0; // 统计新鲜橘子的数量
        
        // 遍历网格，找到所有初始腐烂的橘子，并统计新鲜橘子的数量
        for (int i = 0; i < rows; i++) {
            for (int j = 0; j < cols; j++) {
                if (grid[i][j] == 2) {
                    queue.offer(new int[]{i, j});
                } else if (grid[i][j] == 1) {
                    freshCount++;
                }
            }
        }
        
        // 如果没有新鲜橘子，直接返回0
        if (freshCount == 0) return 0;
        
        int time = 0; // 记录时间
        
        // BFS遍历
        while (!queue.isEmpty()) {
            int size = queue.size();
            boolean isRot = false; // 标记这一轮是否有橘子腐烂
            
            for (int i = 0; i < size; i++) {
                int[] current = queue.poll();
                int row = current[0];
                int col = current[1];
                
                // 检查四个方向
                for (int[] dir : directions) {
                    int newRow = row + dir[0];
                    int newCol = col + dir[1];
                    
                    // 检查是否在网格范围内
                    if (newRow >= 0 && newRow < rows && newCol >= 0 && newCol < cols) {
                        // 如果是新鲜橘子
                        if (grid[newRow][newCol] == 1) {
                            grid[newRow][newCol] = 2; // 标记为腐烂
                            freshCount--; // 新鲜橘子数量减1
                            queue.offer(new int[]{newRow, newCol});
                            isRot = true; // 标记有橘子腐烂
                        }
                    }
                }
            }
            
            // 如果这一轮有橘子腐烂，时间加1
            if (isRot) {
                time++;
            }
        }
        
        // 如果还有新鲜橘子，返回-1，否则返回时间
        return freshCount == 0 ? time : -1;
    }
}
```
