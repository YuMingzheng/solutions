### 解题思路

#### 方法一（取整数）

二分法

```python
class Solution:
    def mySqrt(self, x: int) -> int:
        res = 0
       
        left , right = 0 , x
        while left <= right:
            mid = left + (right - left) // 2
            if mid * mid <= x:
                left = mid + 1
                res = mid
            else:
                right = mid -1
        return res
        
        
```
