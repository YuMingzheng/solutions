### 问题描述

[题目链接](https://leetcode.cn/problems/number-of-islands/description/?envType=study-plan-v2&envId=top-100-liked)

给一个矩阵，`1`表示陆地，`0`表示水，求岛屿的数量

### 解题思路

#### 方法一

深度优先搜索

* 主函数使用`for循环`遍历每一个位置，每找到一个陆地`'1'`，从当前位置开启`DFS`搜索，把搜索到的与该陆地`1`相连的陆地都赋值为`'2'`，表示==已查明==
	* `DFS`：
		* 从当前位置开始沿着四个方向拓展，每到一处复制为`'2'`
		* 边界：`i`或`j`超出范围，或者到了一出水域`'0'`，就return

```java
class Solution {
    public int numIslands(char[][] grid) {
        int ans = 0;
        for(int i = 0; i < grid.length ; i++){
            for(int j = 0 ; j < grid[0].length ; j++){
                if(grid[i][j] == '1'){
                    dfs(grid , i , j);
                    ans ++;
                }
            }
        }
        return ans;
    }

    public void dfs(char[][] grid , int i , int j){
        if(
            i < 0 || i > grid.length -1 || j < 0 || j > grid[0].length -1
            || grid[i][j] != '1'
        ){
            return;
        }
        grid[i][j] = '2'; // '2'表示访问过了
        dfs(grid , i , j - 1);
        dfs(grid , i , j + 1);
        dfs(grid , i + 1 , j);
        dfs(grid , i - 1 , j);
    }
}
```

时间复杂度：$O()$

空间复杂度：$O()$
