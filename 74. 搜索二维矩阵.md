### 问题描述

[题目链接](https://leetcode.cn/problems/search-a-2d-matrix/description/?envType=study-plan-v2&envId=top-100-liked)

从一个每行、每列都是递增排序的二维数组中，判断target值是否在该矩阵中。

### 解题思路

#### 方法一

- 先二分查找第一列，找到该在哪一行中查找
- 确定了哪一行，再在该行中用二分确定是否包含。

```python
class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        def binacySearch(nums, num):
            left, right = 0, len(nums)- 1
            while left <= right:
                mid = (left + right) // 2
                if nums[mid] > num:
                    right = mid -1
                elif nums[mid] < num:
                    left = mid + 1
                else:
                    return mid
            return right + 1
        
        column1 = [_[0] for _ in matrix]
        for i in range(len(column1)):
            if column1[i] == target:
                return True
        column_idx = binacySearch(column1, target)
        row_idx = binacySearch(matrix[column_idx-1], target)
        if row_idx >= len(matrix[0]):
            return False
        return matrix[column_idx-1][row_idx] == target
```
