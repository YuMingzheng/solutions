### 问题描述

[题目链接](https://leetcode.cn/problems/climbing-stairs/description/?envType=study-plan-v2&envId=top-100-liked)

每次可以爬1或2阶台阶，问有多少种可能可以爬到n阶台阶
### 解题思路

#### 方法一

1. 确定dp数组（dp table）以及下标的含义
	1. `dp[i]`表示爬到第i阶有多少方法
2. 确定递推公式
	1. $dp[i] = dp[i-1]+dp[i-2]$
3. dp数组如何初始化
	1. `int[] dp = new int[n+1]`
	2. `dp[0] = 1 , dp[1] = 1`
4. 确定遍历顺序
	1. 一层for循环
5. 举例推导dp数组

```python
class Solution:
    def climbStairs(self, n: int) -> int:
        dp = [0] * n
        if n == 1:
            return 1
        if n == 2:
            return 2
        dp[0], dp[1] = 1, 2
        for i in range(2,n):
            dp[i] = dp[i-1] + dp[i-2]
        return dp[-1] 
```

```java
class Solution {
    public int climbStairs(int n) {
        int[] dp = new int[n+1];
        dp[0] = 1;
        dp[1] = 1;
        if(n < 2){
            return n;
        }
        for(int i = 2 ; i <= n ; i++){
            dp[i] = dp[i-1] + dp[i-2];
        }
        return dp[n];
    }
}
```
