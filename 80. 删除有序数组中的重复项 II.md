### 问题描述

[题目链接]()

给你一个有序数组 `nums` ，请你 **[原地](http://baike.baidu.com/item/%E5%8E%9F%E5%9C%B0%E7%AE%97%E6%B3%95)** 删除重复出现的元素，使得出现次数超过两次的元素**只出现两次** ，返回删除后数组的新长度。

### 解题思路

#### 方法一

重点在`if(nums[ptrL-2]!=nums[ptrR]){`这一段

```java
class Solution {
    public int removeDuplicates(int[] nums) {
        int ptrL = 2;
        for(int ptrR = 2; ptrR < nums.length; ptrR++ ){
            if(nums[ptrL-2]!=nums[ptrR]){
                nums[ptrL] = nums[ptrR];
                ptrL++;
            }
        }
        return ptrL;
    }
}
```

时间复杂度：$O()$

空间复杂度：$O()$
