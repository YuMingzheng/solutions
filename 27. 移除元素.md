### 问题描述

[题目链接](https://leetcode.cn/problems/remove-element/description/?envType=study-plan-v2&envId=top-interview-150)

从`nums`中移除为`val`的元素，并且往前填补空位置

原地算法

### 解题思路

#### 方法一

双指针，左指针手动递增，for遍历右指针，如果遇到非val数字，就令`nums[ptrL]=nums[ptrR]`，然后手动递增左指针。

```java
class Solution {
    public int removeElement(int[] nums, int val) {
        int ptrL=  0;
        for(int ptrR = 0 ; ptrR < nums.length ; ptrR++){
            if(nums[ptrR] != val){
                nums[ptrL]= nums[ptrR];
                ptrL++;
            }
        }
        return ptrL;
    }
}
```

**算法复杂度分析：**

时间复杂度：$O(n)$

空间复杂度：$O(1)$
