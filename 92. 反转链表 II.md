### 问题描述

[题目链接](https://leetcode.cn/problems/reverse-linked-list-ii/description/)

翻转链表指定区间`[left , right]`内

### 解题思路

#### 方法一

1. 引入`p0`指向区间的前一个结点
2. 对区间内的链表翻转
	1. 翻转结束后，`cur`指向区间的后一个节点、`pre`指向区间的`right`
3. 此时`p0.next`仍是`left`的位置，但此时该位置翻转了，应该作为尾巴
	1. 因此`p0.next.next = cur`
	2. `p0.next = pre`

```java
class Solution {
    public ListNode reverseBetween(ListNode head, int left, int right) {
        ListNode dummy = new ListNode(0 , head);
        ListNode p0 = dummy;
        for(int i = 0 ; i < left-1 ; i ++){
            p0 = p0.next;
        }

        ListNode pre = p0;
        ListNode cur = p0.next;
        for(int i = 0 ; i < right - left + 1; i++){
            ListNode temp = cur.next;
            cur.next = pre;
            pre = cur;
            cur=  temp;
        }

        p0.next.next = cur;
        p0.next = pre;

        return dummy.next;
    }
}
```

**算法复杂度分析：**

时间复杂度：$O()$

空间复杂度：$O()$
