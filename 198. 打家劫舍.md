### 问题描述

[题目链接](https://leetcode.cn/problems/house-robber/description/?envType=study-plan-v2&envId=top-100-liked)

一排（首位不成环）房子，不能连续打劫两个相邻的，怎么打劫最多的钱

### 解题思路

#### 方法一

1. 确定dp数组（dp table）以及下标的含义
	1. `int[] dp`表示到第`i`家为止，最多能偷到的钱
2. 确定递推公式
	1. $dp[i] = max(dp[i-1] , dp[i-2] + nums[i])$
3. dp数组如何初始化
	1. `dp[0] = nums[0]`
	2. `dp[1] = max(dp[0],nums[1])`
4. 确定遍历顺序
	1. 从`i=2`开始遍历
5. 举例推导dp数组

```java
class Solution {
    public int rob(int[] nums) {
        int[] dp = new int[nums.length];
        dp[0] = nums[0];
        if(nums.length == 1){
            return dp[0];
        }
        dp[1] = Math.max(nums[0] , nums[1]);
        if(nums.length ==2){
            return dp[1];
        }
        for (int i = 2; i < nums.length; i++) {
            dp[i] = Math.max(dp[i-1] , dp[i-2] + nums[i]);
        }

        return dp[nums.length-1];
    }
}
```

时间复杂度：$O()$

空间复杂度：$O()$
