### 问题描述

[题目链接](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock/?envType=study-plan-v2&envId=top-100-liked)

给一个数组，表示股票的每天价格，选择一天买入、一天卖出，确定最多利润多少。

### 解题思路

#### 方法一

遍历一遍数组，每一次记录一个最小的cost，结果ans为price-cost中的最大值

```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        cost, ans = prices[0], 0
        for price in prices:
            cost = min(cost, price)
            ans = max(price- cost, ans)
        return ans
```

时间复杂度：$O()$

空间复杂度：$O()$
