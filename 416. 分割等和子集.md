### 问题描述

[题目链接](https://leetcode.cn/problems/partition-equal-subset-sum/description/?envType=study-plan-v2&envId=top-100-liked)

判断一个数组nums是否可以分为两个元素和相等的子集。

### 解题思路

#### 方法一

先求和nums，如果是奇数，那么不行；如果是偶数，那么问题转化为：
	是否存在一个子集，使得子集的和是sum(nums)的一半（记为target）
	这就转化为了0-1背包问题

1. 确定dp数组（dp table）以及下标的含义
	1. dp = [False] * (target + 1)
	2. 第i位表示：数组nums中的数组是否存在和为i的子集
2. 确定递推公式
	1. 遍历物品for num in nums
		1. 遍历容量 for c in range(target, num - 1, -1)
			1. dp[c] = dp[c] or dp[c-num]
3. dp数组如何初始化
	1. dp[0] = True
4. 确定遍历顺序
5. 举例推导dp数组

```python
class Solution:
    def canPartition(self, nums: List[int]) -> bool:
        s = sum(nums)
        n = len(nums)
        if s % 2:
            return False
        target = s // 2

        dp = [False] * (target + 1)
        dp[0] = True
        for x in nums:
            for c in range(target, x - 1, -1):
                    dp[c] = dp[c] or dp[c - x]
        return dp[-1]
```

时间复杂度：$O()$
