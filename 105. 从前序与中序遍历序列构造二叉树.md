### 问题描述

[题目链接](https://leetcode.cn/problems/construct-binary-tree-from-preorder-and-inorder-traversal/description/)

如题
- 前序pre和中序ino中均==没有重复元素== 

### 解题思路

#### 方法一

前序`pre`的第一个一定是数的根，找到中序`ino`中根所在的位置，==左边就是左子树、右边就是右子树==，看看左子树有多少个，然后递归pre、ino中对应的左子树、右子树
>java中截断数组，可以用`Arrays.copyOfRange(nums , start , end开区间`

```java
class Solution {
    public TreeNode buildTree(int[] preorder, int[] inorder) {
        int len_pre = preorder.length;
        int len_ino = inorder.length;

        if(len_pre == 0 || len_ino == 0){
            return null;
        }

        TreeNode root = new TreeNode(preorder[0]);
        
        for(int i = 0; i < len_ino ; i++){
            if(preorder[0] == inorder[i]){
                root.left = buildTree(
                    Arrays.copyOfRange(preorder , 1 , i + 1),
                    Arrays.copyOfRange(inorder , 0 , i)
                    );
                root.right = buildTree(
                    Arrays.copyOfRange(preorder , i+ 1 , len_pre),
                    Arrays.copyOfRange(inorder , i+1,len_ino)
                );
                break;
            }
        }

        return root;
    }
}
```
