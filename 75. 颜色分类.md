### 问题描述

[题目链接](https://leetcode.cn/problems/sort-colors/description/?envType=study-plan-v2&envId=top-100-liked)

示例 1：

输入：`nums = [2,0,2,1,1,0]`
输出：`[0,0,1,1,2,2]`

### 解题思路

#### 方法一

统计0,1,2出现的次数
第二次遍历赋值

```python
class Solution:
    def sortColors(self, nums: List[int]) -> None:
        cnt = {0:0,  1:0 , 2:0}
        for num in nums:
            cnt[num] = cnt.get(num) + 1
        idx = 0
        for key , value in cnt.items():
            while value > 0:
                nums[idx] = key
                value -= 1
                idx += 1
        
```

**算法复杂度分析：**

时间复杂度：$O(n)$

空间复杂度：$O(1)$

#### 方法二
>1. 和快排的partition过程类似，小于1的放左边，等于1的放中间，大于1的放右边 
>2. less和more分别为小于区域的右边界和大于区域的左边界 
>3. 索引i从0开始，小于more 
>4. 如果小于1，则要放到小于区域，和小于区域的下一个做交换，同时less++，i++ 
>5. 如果大于1，则要放到大于区域，和大于区域的前一个做交换，同时more--，i不动 
>6. 如果等于1，则是等于区域，less和more都不懂，++i 
>7. 7、当i撞到more时，结束循环

```python
class Solution:
    def sortColors(self, nums: List[int]) -> None:

        i = 0
        left = -1
        right = len(nums)

        while i < right:
            if nums[i] == 0:
                left += 1
                nums[left] , nums[i] = nums[i] , nums[left]
                i += 1
            elif nums[i] == 2:
                right -= 1
                nums[right] , nums[i] = nums[i] , nums[right]
            
            else:
                i += 1
```
