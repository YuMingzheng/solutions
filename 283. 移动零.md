### 问题描述

[题目链接](https://leetcode.cn/problems/move-zeroes/description/)

在保持数组中非零元素原有顺序的前提下，将所有的零移动到数组的末尾。

注意：只能在原数组上进行操作。

### 解题思路

#### 方法一

- 使用两个指针 `i` 和 `j`，初始时都指向数组的首元素，其中 `i` 指向的是 0，`j` 指向的是非零元素。
- 当 `j` 指向非零元素时，交换 `i` 和 `j` 位置的元素，并同时将两个指针向后移动。
- 如果 `j` 指向的是零，则 `j` 向后移动，`i` 保持不动，且不进行交换。`i` 和 `j` 之间的元素都为零。
```python
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        i , j = 0, 0
        while j < len(nums):
            if nums[j] != 0:
                nums[i], nums[j] = nums[j] , nums[i]
                i += 1
            j += 1
        
```

```java
class Solution {
    public void moveZeroes(int[] nums) {
        int i = 0;
        int j = 0;
        while (j < nums.length) {
            if (nums[j] != 0) {
                int temp = nums[i];
                nums[i] = nums[j];
                nums[j] = temp;
                i++;
            }
            j++;
        }
    }
}
```

**算法复杂度分析：**

时间复杂度：$O(n)$

空间复杂度：$O(1)$
