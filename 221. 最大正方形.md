### 问题描述

[题目链接](https://leetcode.cn/problems/maximal-square/description/?envType=problem-list-v2&envId=2cktkvj)

在二维数组中找到只包含1的最大正方形，返回其面积。

### 解题思路

#### 方法一

1. 确定dp数组（dp table）以及下标的含义
	1. 二维数组，$dp[i][j]$表示以$i,j$为右下角，且只包含1的正方形的边长最大值
2. 确定递推公式
	1. 对于位置$i,j$
		1. 如果该位置是0
			1. 则$dp[i][j]=0$
		2. 如果该位置是1
			1. $dp[i][j]=min(dp[i-1][j], dp[i-1][j-1], dp[i][j-1])+1$ 
3. dp数组如何初始化
4. 确定遍历顺序
5. 举例推导dp数组

```python
class Solution:
    def maximalSquare(self, matrix: List[List[str]]) -> int:
        if len(matrix) == 0 or len(matrix[0]) == 0:
            return 0
        maxSide = 0
        rows, columns = len(matrix), len(matrix[0])
        dp = [[0] * columns for _ in range(rows)]

        for i in range(rows):
            for j in range(columns):
                if matrix[i][j] == '1':
                    if i == 0 or j == 0:
                        dp[i][j] = 1
                    else:
                        dp[i][j] = min(dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1]) + 1
                    maxSide = max(maxSide, dp[i][j])
        maxSquare = maxSide * maxSide
        return maxSquare
```

时间复杂度：$O()$
