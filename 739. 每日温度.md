### 问题描述

[题目链接]([739. 每日温度 - 力扣（LeetCode）](https://leetcode.cn/problems/daily-temperatures/description/?envType=problem-list-v2&envId=2cktkvj))

给一个表示每一天温度的数组，找到下一天更高温度出现在第几天后。

### 解题思路

#### 方法一

（一段话解释算法的逻辑）

```python
class Solution:
    def dailyTemperatures(self, temperatures: List[int]) -> List[int]:
        ans = [0 for _ in temperatures]
        stack = []
        for i in range(len(temperatures)):
            curr = temperatures[i]
            while stack and curr > temperatures[stack[-1]]:
                prev_index = stack.pop()
                ans[prev_index] = i - prev_index
            stack.append(i)
        return ans
        
```

**算法复杂度分析：**

时间复杂度：$O(n)$

空间复杂度：$O(n)$
